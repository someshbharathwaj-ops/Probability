Phase 0 Lesson 0.1 Foundations → Randomness

What is Randomness,
exactly?

Sample Space · Events · Random Variables · Probability Axioms
🔥 Why Humans Invented This

Around 1654, Pascal and Fermat wrote letters to each other about gambling disputes. They were the first to ask: how do you reason about outcomes you cannot know in advance?

Three centuries later, Kolmogorov gave probability its axiomatic foundation — not for gambling, but because the universe itself is random. Quantum mechanics, thermal noise, genetics, financial markets, neural activity — all fundamentally uncertain.

Probability is the language we use to reason precisely about imprecise things. Without it, we'd be silent about everything that matters most.

🧠 The Core Objects

A random experiment is any process whose outcome we cannot predict. A sample space Ω is the set of all possible outcomes. An event A is a subset of Ω — outcomes we care about.

A random variable X is a function X: Ω → ℝ. It's not a variable — it's a measurement machine. You plug in an outcome ω, you get a number X(ω). This distinction is crucial.

⚙️ Kolmogorov's Three Axioms — All of Probability Theory Flows From These

Kolmogorov (1933) defined probability as any function P: events → [0,1] satisfying:

AXIOM 1: $P(A) \geq 0$ for all events $A$

AXIOM 2: $P(\Omega) = 1$

AXIOM 3: $P(A \cup B) = P(A) + P(B)$ if $A \cap B = \emptyset$

That's it. Everything else — conditional probability, independence, expectation, variance — is a theorem derived from these three rules. They are not arbitrary: they capture exactly what a "proportion" should mean.

⚠
Permanent Trap to Eliminate
A random variable X is not "a variable that takes random values." It is a function X: Ω → ℝ. Writing "let X be random" means: let X be this function defined on the sample space. The randomness comes from the random choice of ω, not from X itself changing.
Phase 0 Lesson 0.2

The Shape of Uncertainty:
PMF & PDF

Probability Mass Function · Probability Density Function · Distribution
🔥 Why This Exists

Knowing a random variable exists isn't enough. We need to know where its probability lives. Is it equally spread? Concentrated near 0? Symmetric? The distribution function answers: "How is probability distributed across values?"

🧠 PMF — Discrete Case

For discrete X, the Probability Mass Function is $p(x) = P(X = x)$. Think of probability as physical weight sitting at discrete points on the number line.

Validity conditions: (1) $p(x) \geq 0$ everywhere, and (2) $\sum_x p(x) = 1$. All the weight must sum to one.

PMF Simulator — Theory vs Sampling
Indigo bars = true PMF (theoretical). Amber markers = empirical sample frequencies. As n→∞, empirical → theoretical. This convergence is the essence of statistics.
🧠 PDF — Continuous Case

For continuous X, $P(X = x) = 0$ for any specific x. Probability only makes sense over intervals. The Probability Density Function f(x) satisfies:

$$P(a \leq X \leq b) = \int_a^b f(x)\, dx$$

Critical: f(x) is not a probability — it is a density. f(x) can be greater than 1! Only areas (integrals) give probabilities. The total area under f must equal 1: $\int_{-\infty}^{\infty} f(x)\,dx = 1$.

Phase 0 Lesson 0.3

Expectation:
The Center of Mass

Expected Value · Linearity · Law of the Unconscious Statistician
🔥 Why This Exists

If you run an experiment 10,000 times and average the results, what do you get? Engineers need this number to design systems. Insurers need it to set premiums. Physicists need it to predict particle behavior. Expectation is the single best representative number for a random variable.

🧠 Definition — Balance Point of Probability

Imagine probability mass as physical weight on a number line. The expectation is the balancing point — the fulcrum that keeps the distribution in equilibrium.

$$\mathbb{E}[X] = \sum_x x \cdot p(x) \quad \text{(discrete)}$$ $$\mathbb{E}[X] = \int_{-\infty}^{\infty} x \cdot f(x)\, dx \quad \text{(continuous)}$$
Expectation Balance Beam

The amber triangle ▲ is the fulcrum — it sits exactly at E[X]. Watch it slide as you change the distribution. The taller bars are "heavier" and pull the balance point toward them.
⚙️ The Four Properties You Must Know
  • Linearity: $\mathbb{E}[aX + b] = a\mathbb{E}[X] + b$ — always, unconditionally.
  • Superposition: $\mathbb{E}[X + Y] = \mathbb{E}[X] + \mathbb{E}[Y]$ — even if X, Y are dependent!
  • Constants: $\mathbb{E}[c] = c$ for any constant c.
  • LOTUS: $\mathbb{E}[g(X)] = \sum g(x)p(x)$ — don't find the distribution of g(X) first.
💡 The Most Underrated Miracle

$\mathbb{E}[X+Y] = \mathbb{E}[X]+\mathbb{E}[Y]$ holds even when X and Y are dependent. Most students think independence is required — it is not. This linearity is the reason probabilistic arguments can decompose complex systems into simple parts. It is the engine behind most probability computations.

Phase 0 Lesson 0.4

Variance:
The Energy of Uncertainty

Variance · Standard Deviation · Computational Formula
🔥 Why This Exists

Expectation is the center, but two distributions with the same center can be completely different. One might be almost always near the mean; another wildly spread. Variance measures how energetically the distribution fluctuates. In physics: variance is literally the kinetic energy of random fluctuations.

🧠 Definition and the Computational Trick
$$\text{Var}(X) = \mathbb{E}[(X-\mu)^2] = \mathbb{E}[X^2] - (\mathbb{E}[X])^2$$

The second form is the computational formula — always faster to use. The standard deviation $\sigma = \sqrt{\text{Var}(X)}$ has the same units as X, making it interpretable.

⚙️ Key Properties
  • Scaling: $\text{Var}(aX+b) = a^2\text{Var}(X)$ — shifts don't affect spread; scaling squares it.
  • Independent sum: $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y)$ only if X⊥Y.
  • Non-negativity: $\text{Var}(X) \geq 0$, with equality iff X is a constant a.s.
⚠
The Most Common Error in Applied Probability
Var(X+Y) ≠ Var(X)+Var(Y) in general. The correct formula is Var(X+Y) = Var(X)+Var(Y)+2Cov(X,Y). Independence forces Cov=0, giving the simple sum. Forgetting this causes errors in noise analysis, portfolio theory, and signal processing.
Phase 0 Lesson 0.5

The Chebyshev Inequality:
Universal Safety Net

Distribution-Free Bounds · Tail Probability · Law of Large Numbers
🔥 Why This Exists

Often you know only μ and σ — not the full distribution. A system designer needs to know: "how often will the signal exceed 5 volts?" without knowing the exact noise distribution. Chebyshev gives a universal upper bound valid for any distribution — the price is conservatism, the reward is universality.

🧠 The Statement
$$P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2}$$

Reading this: "The probability that X deviates from its mean by more than k standard deviations is at most 1/k²." At k=2: at most 25%. At k=3: at most 11%. At k=10: at most 1%.

The proof uses only two lines of algebra — essentially, variance bounds tail weight by definition.

Chebyshev Bound vs Actual — How Conservative?
2.0
Chebyshev bound: 1/k² = 0.250 | Actual tail: 0.046 | Chebyshev is 5.4× looser
Red shaded region = actual tail probability. Red dashed lines = ±kσ from mean. The bound is always above the actual — this is the guarantee. For Gaussian it's very loose; for worst-case distributions it's tight.
📐 The Two-Line Proof

Let A = {|X−μ| ≥ ε}. Then $(X-\mu)^2 \geq \varepsilon^2$ whenever $\omega \in A$. Therefore:

$$\sigma^2 = \mathbb{E}[(X-\mu)^2] \geq \mathbb{E}[(X-\mu)^2 \cdot \mathbf{1}_A] \geq \varepsilon^2 \cdot P(A)$$

Dividing: $P(|X-\mu| \geq \varepsilon) \leq \sigma^2/\varepsilon^2$. Setting ε = kσ gives 1/k². This is the entire proof — beautiful in its simplicity.

Phase 0 Lesson 0.6

The CDF & Variable
Transformations

Cumulative Distribution Function · Monotone Transforms · Jacobian
🧠 CDF — Accumulated Probability
$$F_X(x) = P(X \leq x) = \int_{-\infty}^x f(t)\,dt$$

The CDF grows from 0 to 1 as x sweeps −∞ to +∞. It's right-continuous and non-decreasing. Key: $f(x) = F'(x)$ wherever F is differentiable.

Power: $P(a < X \leq b) = F(b) - F(a)$. Any probability query on X reduces to evaluating F.

⚙️ Transformation of Variables

If Y = g(X) where g is monotone increasing: the CDF method gives $F_Y(y) = F_X(g^{-1}(y))$. Differentiating: the PDF of Y is

$$f_Y(y) = f_X(g^{-1}(y)) \cdot \left|\frac{dg^{-1}}{dy}\right|$$

The Jacobian factor $|dg^{-1}/dy|$ corrects for stretching/compressing of probability mass. Intuition: probability is conserved, so $f_X(x)|dx| = f_Y(y)|dy|$, giving the Jacobian.

Let X have PDF $f(x) = \frac{1}{2}x^2 e^{-x}$ for $x \geq 0$, zero otherwise. (a) Verify f is a valid PDF. Identify the distribution family. Identify the Distribution
$\int_0^\infty x^2 e^{-x} dx = \Gamma(3) = 2! = 2$. So $(1/2)\cdot 2 = 1$ ✓. This is a Gamma(3,1) distribution (shape k=3, rate 1). For Gamma(n,1): E[X]=n, Var(X)=n by the moment formulas.
2
Moments via Gamma Function
$\mathbb{E}[X] = \frac{1}{2}\int_0^\infty x^3 e^{-x}dx = \frac{\Gamma(4)}{2} = \frac{6}{2} = \mathbf{3}$
$\mathbb{E}[X^2] = \frac{\Gamma(5)}{2} = \frac{24}{2} = 12$
$\text{Var}(X) = 12 - 9 = \mathbf{3}$, so $\sigma = \sqrt{3}$

PDF of Y = e^{−X} via CDF Method
Support: X≥0 → Y = e^{-X} ∈ (0,1]. CDF: $F_Y(y) = P(Y≤y) = P(e^{-X}≤y) = P(X ≥ -\ln y) = 1 - F_X(-\ln y)$. Differentiating: $f_Y(y) = f_X(-\ln y) \cdot \frac{1}{y}= \frac{1}{2}(-\ln y)^2 e^{\ln y} \cdot \frac{1}{y} = \frac{(-\ln y)^2}{2}$ for $y\in(0,1]$. Elegant: the log-transform converts Gamma to this "log-Gamma" shape.
4
Chebyshev Bound — The Deep Reasoning
$P(Y \leq e^{-6}) = P(e^{-X} \leq e^{-6}) = P(X \geq 6)$ (since $e^{-x}$ is decreasing). Now: $P(X \geq 6) = P(X - 3 \geq 3) \leq P(|X-3| \geq 3)$ (one-sided is weaker than two-sided). By Chebyshev: $P(|X-3| \geq 3) \leq \frac{\text{Var}(X)}{3^2} = \frac{3}{9} = \mathbf{1/3}$. Key insight: we translated the event about Y back to an event about X — the distribution we understand — then applied Chebyshev.

Phase 1 Lesson 1.1

Joint Distributions:
Probability as Terrain

Joint PDF · 2D Probability Mountains · Bivariate Gaussian
🔥 Why We Need This

Real systems have many quantities that interact. Temperature and pressure. Signal and noise. The input and output of a system. To model their interaction, probability must live in two or more dimensions simultaneously. A single marginal distribution tells you about each alone — but misses the relationship between them.

🧠 Joint PDF — Probability as Mountain Range

The joint PDF $f(x,y)$ describes a probability surface over the $(x,y)$ plane. High peaks = regions of high probability density. The total volume under the surface = 1.

$$P((X,Y) \in A) = \iint_A f(x,y)\,dx\,dy$$

The marginal PDF of X is obtained by collapsing the y-dimension: $f_X(x) = \int_{-\infty}^\infty f(x,y)\,dy$. Geometrically: you're projecting the mountain onto the x-axis.

Phase 1 Lesson 1.2

Conditional Distributions:
Knowledge Changes Everything

Conditional PDF · Bayes' Theorem · Law of Total Expectation
🔥 The Central Problem of Inference

You observe that X = 3. How does this update your beliefs about Y? If X and Y are correlated, knowing X moves the entire distribution of Y. Conditional distributions capture this belief update — they are the mathematical language of inference, from radar to machine learning.

🧠 Conditional PDF — Slicing the Mountain
$$f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)}$$

Geometrically: take a vertical slice of the joint density surface at X=x, then normalize it so the slice integrates to 1. The result is a valid PDF in y for each fixed x.

💡 Law of Total Expectation — The Towering Theorem
$$\mathbb{E}[Y] = \mathbb{E}[\mathbb{E}[Y|X]]$$

Compute the conditional expectation of Y given X = x (for each x), then average over X. This "tower property" is the foundation of Bayesian computation, hidden Markov models, and all of statistical decision theory.

Phase 1 Lesson 1.3

Independence:
When Variables Ignore Each Other

Statistical Independence · Covariance · The ρ=0 Trap
🧠 The Definition
$$X \perp Y \iff f(x,y) = f_X(x) \cdot f_Y(y) \quad \forall x,y$$

Equivalently: $f(y|x) = f_Y(y)$ — the conditional equals the marginal. Knowing X tells you nothing about Y.

Key consequences: $\mathbb{E}[XY]=\mathbb{E}[X]\mathbb{E}[Y]$, $\text{Var}(X+Y)=\text{Var}(X)+\text{Var}(Y)$, $F(x,y)=F_X(x)F_Y(y)$.

🧠 Covariance & Correlation
$$\text{Cov}(X,Y) = \mathbb{E}[XY] - \mathbb{E}[X]\mathbb{E}[Y] \qquad \rho = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$$

$\rho \in [-1,1]$. ρ=1: perfect positive linear relationship. ρ=0: uncorrelated. ρ=−1: perfect negative linear relationship.

💀
The Most Important Trap in All of Probability
Uncorrelated does NOT mean independent. Let X ~ Uniform(−1,1), Y = X². Then Cov(X,Y) = E[X³] − E[X]E[X²] = 0 (since E[X³]=0). So ρ=0 — they are uncorrelated. But knowing X tells you exactly Y = X² — they are completely dependent! Zero correlation only detects linear relationships.
💡 The Hierarchy
  • Independent → Uncorrelated (always)
  • Uncorrelated → Independent (ONLY for jointly Gaussian distributions)
  • For all other distributions: uncorrelated ≠ independent
Phase 1 Lesson 1.4

Transformations:
Changing the Measurement

Jacobian · Multivariate Transforms · Conservation of Probability
🧠 The Jacobian Formula

If $(X_1,X_2) \to (Y_1,Y_2)$ via a smooth invertible transformation, the joint PDF transforms as:

$$f_{Y_1,Y_2}(y_1,y_2) = f_{X_1,X_2}(g_1^{-1},g_2^{-1}) \cdot \left|\det\begin{pmatrix}\partial x_1/\partial y_1 & \partial x_1/\partial y_2 \\ \partial x_2/\partial y_1 & \partial x_2/\partial y_2\end{pmatrix}\right|$$

The Jacobian determinant measures how the transformation stretches area elements. Probability must be conserved: $f_{X}|dx| = f_Y|dy|$. The Jacobian is just this conservation law in matrix form.

💡 Classic Example: Cartesian to Polar

$(X,Y) \to (R,\Theta)$ where $R=\sqrt{X^2+Y^2}$, $\Theta=\arctan(Y/X)$. The Jacobian is $r$ (the radial factor). If X,Y are independent standard normals: $f(x,y) = \frac{1}{2\pi}e^{-(x^2+y^2)/2}$. In polar: $f(r,\theta) = \frac{1}{2\pi}e^{-r^2/2} \cdot r$. So $R^2 \sim \text{Exponential}(1/2)$, which is $\chi^2(2)$. This is why $R=\sqrt{-2\ln U}$ generates Rayleigh-distributed radii in the Box-Muller algorithm.

⚡ Boss Problem — Phase 1
Let $(X,Y)$ be jointly Gaussian with zero means, unit variances, and correlation $\rho$. (a) Let U = X+Y, V = X−Y. Find Cov(U,V). Are U and V independent? (b) Find E[X | U = u]. (c) Let $W = X^2 + Y^2$. For $\rho = 0$, find the PDF of W and name its distribution. (d) For general ρ, compute Var(U) and find when Var(U) > Var(V).
1
Covariance of U and V
Cov(U,V) = Cov(X+Y, X−Y) = Var(X) − Var(Y) = 1 − 1 = 0. Since (U,V) are a linear transformation of Gaussians, they're jointly Gaussian. Zero covariance among Gaussians → independence! U⊥V.
2
Conditional Expectation
X = (U+V)/2. E[X|U=u] = E[(U+V)/2 | U=u] = (u + E[V|U=u])/2 = (u + 0)/2 = u/2, since U⊥V implies E[V|U]=E[V]=0.
3
Distribution of W = X² + Y² (ρ=0)
With ρ=0: X,Y are independent standard normals. In polar coordinates, $f(r,\theta) = (1/2\pi)e^{-r^2/2}\cdot r$. So $f_R(r) = re^{-r^2/2}$ (Rayleigh). Then $W = R^2$ has CDF $P(W\leq w) = P(R\leq\sqrt{w}) = 1-e^{-w/2}$. This is Exponential(1/2), also called χ²(2). Elegant: the sum of squares of two independent normals is exponential.
4
Variance and the Condition
Var(U) = Var(X+Y) = Var(X)+Var(Y)+2Cov(X,Y) = 1+1+2ρ = 2+2ρ. Var(V) = 2−2ρ. Var(U) > Var(V) iff 2+2ρ > 2−2ρ iff 4ρ > 0 iff ρ > 0. When variables are positively correlated, their sum is more variable than their difference.

Phase 2Lesson 2.1

From Random Variable
to Random Process

The Conceptual Leap · Time Indexing · Sample Functions
🔥 The Fundamental Upgrade

A random variable gives you one number per experiment. But most real phenomena evolve over time. The voltage noise in a circuit across 1 millisecond. The temperature at every moment of a day. The price of a stock over a year. These are not numbers — they are functions.

A random process X(t) assigns a random function of time to each experiment. It is simultaneously: (1) for each fixed t, a random variable; and (2) for each fixed ω, a deterministic function of t called a sample function or realization.

RV vs Random Process — The Visual Difference
RV mode: each experiment (color) produces ONE number on the line. Process mode: each experiment produces an ENTIRE TRAJECTORY. The shift from point to function is the birth of random process theory.
🧠 Formal Definition

A random process is a function $X: T \times \Omega \to \mathbb{R}$ where T is the index set (usually time) and $(\Omega, \mathcal{F}, P)$ is a probability space. We write $X(t)$ suppressing ω, but ω is always implicit.

Fix ω → vary t
Sample function X(t,ω): a deterministic signal. What you observe in one run of the experiment.
Fix t → vary ω
Random variable X(t,·): the distribution of the process at one instant. Has its own mean and variance.
Phase 2Lesson 2.2

The Ensemble:
Infinite Parallel Universes

Ensemble Averages · Time Averages · Ergodicity Preview
🧠 Two Ways to Measure a Process

The ensemble is the collection of all possible sample functions — one per possible outcome ω. Think: infinitely many parallel universes, each running the same random experiment.

Ensemble Average (theoretical)
Fix time t. Average X(t,ω) across all ω. Gives μ_X(t) = E[X(t)]. This is what we compute mathematically.
Time Average (measured)
Fix ω. Average X(t,ω) across time t. This is what you measure from one signal in the lab. May or may not equal the ensemble average.
Ensemble Visualizer — See All Parallel Universes
Each colored curve = one sample function (one parallel universe). White dashed = ensemble mean μ(t). Gold band = ±σ uncertainty cloud. Watch: Wiener mean stays at 0 but spread grows. OU is "pulled back" — mean-reverting.
💡 Ergodicity — The Bridge Between Theory and Measurement

A process is ergodic if time averages equal ensemble averages. For ergodic processes, you can estimate μ_X from a single long observation. For non-ergodic processes (like a constant random variable), one sample tells you almost nothing about the ensemble. Most stationary processes of engineering interest are ergodic.

Phase 2Lesson 2.3

Mean & Variance
Functions of a Process

μ_X(t) · σ²_X(t) · Autocorrelation Function Preview
🧠 They Are Functions of Time
MEAN FUNCTION: $\mu_X(t) = \mathbb{E}[X(t)]$ — deterministic function of t

VARIANCE FUNCTION: $\sigma^2_X(t) = \mathbb{E}[(X(t)-\mu_X(t))^2]$ — spread at each instant

AUTOCORRELATION: $R_X(t_1,t_2) = \mathbb{E}[X(t_1)X(t_2)]$ — memory between two instants

The autocorrelation $R_X(t_1,t_2)$ is the key new object. It tells you: if the process is high at time $t_1$, is it likely high at time $t_2$? This encodes the temporal memory of the process. Note: $R_X(t,t) = \mathbb{E}[X^2(t)]$ = second moment (total instantaneous power).

💡 Why ACF Matters More Than Mean

For a zero-mean process, μ(t)=0 tells you nothing about the signal's structure. The autocorrelation tells you everything else — how fast it fluctuates, what its frequency content is, whether it has memory. In fact, for Gaussian processes, the ACF combined with the mean fully specifies the entire process — all finite-dimensional distributions.

Phase 3Lesson 3.1

The Full Distribution
of a Random Process

Finite-Dimensional Distributions · Gaussian Processes · Kolmogorov Extension
🔥 The Problem of Complete Description

A random variable is described by one distribution. A random process at N different times requires an N-dimensional joint distribution. To fully specify a random process, you need to know all finite-dimensional distributions simultaneously — for every possible choice of times and every N.

🧠 Finite-Dimensional Distributions

The complete specification: for any $n$ and times $t_1 < \cdots < t_n$, we need:

$$F_{t_1,\ldots,t_n}(x_1,\ldots,x_n) = P(X(t_1)\leq x_1,\ldots,X(t_n)\leq x_n)$$

These must be consistent (Kolmogorov's extension theorem guarantees the converse: consistent families always define a process). In practice, specifying infinitely many distributions is impossible — so we look for processes with special structure.

💡 Why Gaussian Processes Are Special

For a Gaussian process, every finite-dimensional distribution is multivariate Gaussian. A multivariate Gaussian is fully specified by its mean vector and covariance matrix. Therefore: the entire Gaussian process is specified by just two functions: $\mu(t)$ and $R(t_1,t_2)$. This is why Gaussian processes are the workhorses of machine learning — complex models, but only two functions to specify.

Phase 3Lesson 3.2

Stationarity:
Time-Shift Invariance

SSS · WSS · Engineering Significance · Geometric Interpretation
🔥 Why Stationarity Is the Key Assumption

Most physical noise sources in steady state have constant statistical character — thermal noise in a resistor today is statistically identical to thermal noise yesterday. This time-translation invariance is stationarity. It makes the mathematics tractable and is the foundation of all signal processing theory.

🧠 Two Levels of Stationarity
SSS (Strict Sense Stationary):
All finite-dimensional distributions are shift-invariant: $f_{X(t_1+\tau),\ldots,X(t_n+\tau)} = f_{X(t_1),\ldots,X(t_n)}$ for all $n,\tau$

WSS (Wide Sense Stationary):
(1) $\mathbb{E}[X(t)] = \mu$ (constant) and (2) $R_X(t_1,t_2) = R_X(t_1-t_2)$ (depends only on lag $\tau = t_1-t_2$)

WSS is weaker than SSS. For Gaussian processes, WSS implies SSS — this is why Gaussian processes are so convenient. For others: SSS → WSS, but WSS does not generally imply SSS.

Stationarity Lab — Spot the Difference
WSS: mean (white dashed) is constant, spread (gold band) is constant across time. Non-stationary: mean drifts OR variance grows OR both. The ensemble snapshot at t₁ looks different from t₂.
⚙️ Properties of R_X(τ) for WSS
  • R_X(0) = E[X²(t)] — total average power of the process.
  • R_X(τ) = R_X(−τ) — even function (Hermitian for complex).
  • |R_X(τ)| ≤ R_X(0) — maximum at zero lag (by Cauchy-Schwarz).
  • R_X(τ) is positive semi-definite — its Fourier transform is non-negative (= PSD).
  • R_X(τ) → μ² as τ → ∞ (if process loses memory).
⚠
The Most Misunderstood Fact About Stationarity
WSS does NOT mean each sample function looks the same at all times. A single realization still varies wildly — it's a random signal! WSS means: if you photographed the entire ensemble at t=3 and t=7, the two photos would be statistically identical. The ensemble is stationary; individual paths are not.
Phase 3Lesson 3.3

Autocorrelation:
The Memory of Randomness

R_X(τ) · Memory Length · Sinusoidal Interpretation · Engineering Meaning
🔥 Why Memory Matters

Does the noise at t=0 "remember" where it was at t=−0.001? In a telephone cable, the noise sample at one moment bleeds into the next — causing intersymbol interference. In weather, temperature tomorrow depends on temperature today. In finance, momentum means past returns predict future returns. Autocorrelation is the mathematical measure of how much a process remembers its own past.

🧠 Definition for WSS
$$R_X(\tau) = \mathbb{E}[X(t) \cdot X(t+\tau)]$$

Read: "multiply the process with a time-shifted copy of itself, then take the ensemble average." For WSS, this depends only on the lag τ, not on t.

At τ=0: $R_X(0) = \mathbb{E}[X^2(t)]$ = total average power. As τ→∞: if memory fades, $R_X(\tau) \to \mu^2$ (for zero mean: → 0). The correlation time is the lag at which R_X has decayed significantly — it measures how "long" the process's memory lasts.

💡 Physical Meaning of ACF Shapes
  • $R(\tau) = \sigma^2 e^{-\lambda|\tau|}$: Markov/Ornstein-Uhlenbeck. RC circuit with white noise input. Correlation time = 1/λ.
  • $R(\tau) = \sigma^2 e^{-\lambda|\tau|}\cos(\omega_0\tau)$: Damped LC resonance. Bandpass noise around ω₀.
  • $R(\tau) = \sigma^2\,\text{sinc}(2W\tau)$: Bandlimited white noise. Correlated over time 1/(2W).
  • $R(\tau) = \sigma^2\delta(\tau)$: White noise — no memory at all. Theoretically ideal, physically unrealizable.
Phase 3Lesson 3.4

Power Spectral Density:
The Frequency Portrait

Wiener-Khinchin Theorem · PSD Properties · Linear Filtering
🔥 Why the Frequency Domain Matters

A random signal carries energy spread across all frequencies. A slowly varying noise (low frequency) is different from a rapidly oscillating noise (high frequency) even if both have the same total power. The PSD is the energy spectrum — it tells you which frequencies are loud and which are quiet. Engineers use it to design filters that suppress noise at unwanted frequencies.

🧠 Wiener-Khinchin Theorem
$$S_X(f) = \int_{-\infty}^{\infty} R_X(\tau)\, e^{-j2\pi f\tau}\, d\tau \qquad \Leftrightarrow \qquad R_X(\tau) = \int_{-\infty}^{\infty} S_X(f)\, e^{j2\pi f\tau}\, df$$

The PSD is the Fourier transform of the autocorrelation. Setting τ=0: $R_X(0) = \int S_X(f)\,df$ = total average power. The PSD literally shows how power is distributed across frequencies.

Properties: $S_X(f) \geq 0$ always (non-negative). $S_X(f) = S_X(-f)$ (even, for real processes). These are guaranteed by the positive semi-definiteness of R_X(τ).

⚙️ Through a Linear Filter — The Engineering Formula

If WSS process X(t) passes through an LTI filter with frequency response H(f):

$$S_Y(f) = |H(f)|^2\, S_X(f)$$

The output PSD = input PSD × |H(f)|². This is the single most powerful result in random signal processing. Design H(f) to shape the output noise spectrum: low-pass filter removes high-frequency noise; bandpass filter isolates a signal band.

Phase 3Lesson 3.5

The Poisson Process:
Pure Randomness in Time

Counting Process · Memoryless Property · Interarrival Times
🔥 Where This Appears

Photons hitting a detector. Customers arriving at a bank. Network packets arriving at a router. Radioactive decays. Neuron firing events. All of these are events scattered randomly in time. The Poisson process is the canonical model — the most purely random way to scatter events in time with a given average rate.

🧠 Definition via Three Axioms
  • Independent increments: Events in non-overlapping time intervals are independent of each other.
  • Stationary increments: The distribution of events in [t, t+s] depends only on the length s, not the start time t.
  • No simultaneous events: P(2+ events in [t, t+dt]) = o(dt) as dt→0.
$$P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}, \quad k = 0,1,2,\ldots$$

These axioms force the Poisson distribution — it is not assumed, it is derived. The parameter λ is the rate (events per unit time).

Poisson Process Universe — Watch Events Arrive
4.0
N(t) = 0 | E[N(t)] = λt = 0.0 | Var[N(t)] = λt = 0.0
Green staircase = N(t) count. Amber dashed = expected count λt. Each vertical dash = one event. Notice E[N]=Var[N]=λt — the signature of Poisson randomness.
🧠 Interarrival Times — The Memoryless Property
$$T_k \sim \text{Exponential}(\lambda) \qquad P(T > t) = e^{-\lambda t}$$

The time between successive events follows an exponential distribution. The exponential distribution has the memoryless property:

$P(T > s+t \mid T > s) = P(T > t)$

No matter how long you've already waited, the remaining waiting time has the same distribution. It's as if the process has no memory of your wait.

💡 The Beautiful Identity: Mean = Variance

For N(t) ~ Poisson(λt): E[N(t)] = Var[N(t)] = λt. They are equal! This "index of dispersion = 1" is the defining signature of Poisson randomness. If you observe a counting process and find Mean = Variance, you're looking at Poisson structure. More dispersion (variance > mean) suggests clustering; less dispersion suggests regularity.

⚡⚡ GRAND MONSTER PROBLEM
Let X(t) be a WSS random process with zero mean and autocorrelation $R_X(\tau) = 4e^{-2|\tau|}\cos(3\tau) + 9$.

Part A. Is R_X(τ) a valid autocorrelation function? Find E[X²(t)] and S_X(f).

Part B. The process passes through an ideal LPF with cutoff $f_c = 1$ Hz, yielding Y(t). Find S_Y(f), R_Y(τ), and E[Y²(t)].

Part C. Define Z(t) = X(t) − X(t − t₀). Find R_Z(τ), E[Z²(t)], and find the value of t₀ that minimizes E[Z²].

Part D. Using Chebyshev, find the minimum k such that P(|X(t)| ≥ k) ≤ 0.1.

Part E (Conceptual). The Poisson process N(t) with rate λ satisfies R_N(t₁,t₂) = λ·min(t₁,t₂) + λ²t₁t₂. Is N(t) WSS? Justify precisely and compute Var(N(t)) using R_N.
A
Validity, Total Power, PSD
Validity: R_X(τ) must be (1) even: R_X(τ)=R_X(−τ) ✓ (since e^{−2|τ|} and cos(3τ) are both even). (2) Positive semi-definite: equivalently S_X(f) ≥ 0. We check via computing S_X.

Total power: E[X²(t)] = R_X(0) = 4·1·1 + 9 = 13 W

PSD: S_X(f) = ℱ{4e^{−2|τ|}cos(3τ)} + ℱ{9} Using ℱ{e^{−a|τ|}} = 2a/(a²+(2πf)²) and frequency shift:
$S_X(f) = \frac{4}{\pi}\left[\frac{2}{4+(2\pi f-3)^2} + \frac{2}{4+(2\pi f+3)^2}\right] + 9\delta(f)$
S_X(f) ≥ 0 confirmed (sum of Lorentzians + delta). Valid ✓. The constant +9 in R_X means the process has a DC component with power 9 W — wait! The problem states zero mean, but R_X(∞) = 9 ≠ μ² = 0. Contradiction resolved: R_X(τ) → 9 as τ→∞ means μ² = 9, so μ = ±3. We revise: E[X(t)] = 3 (taking positive root). The 9δ(f) spike in S_X carries the DC power.
B
LPF Filtering — Power After the Filter
Ideal LPF: H(f) = 1 for |f| ≤ 1 Hz, 0 otherwise. So S_Y(f) = S_X(f)·rect(f/2).

E[Y²] = ∫_{−1}^{1} S_X(f) df = 9 (DC spike at f=0 passes through) + ∫_{−1}^{1} \frac{8}{\pi}\left[\frac{1}{4+(2πf-3)^2}+\frac{1}{4+(2πf+3)^2}\right] df

The center frequencies ±3/(2π) ≈ ±0.477 Hz are within |f| ≤ 1, so the filter passes most of the AC power near those peaks. The integral evaluates to approximately 4·(2/π)·arctan(... ) ≈ 3.2 W. Total E[Y²] ≈ 12.2 W. The LPF removed approximately 0.8 W of AC power (high-frequency tails).
C
Difference Process Z(t) = X(t) − X(t−t₀)
$R_Z(\tau) = \mathbb{E}[Z(t)Z(t+\tau)] = 2R_X(\tau) - R_X(\tau+t_0) - R_X(\tau-t_0)$

$\mathbb{E}[Z^2] = R_Z(0) = 2R_X(0) - 2R_X(t_0) = 2(13) - 2(4e^{-2t_0}\cos(3t_0)+9)$
$= 8 - 8e^{-2t_0}\cos(3t_0)$

Minimum over t₀ > 0: Minimize $g(t_0) = -e^{-2t_0}\cos(3t_0)$, i.e., maximize $h(t_0) = e^{-2t_0}\cos(3t_0)$.
$h'(t_0) = e^{-2t_0}(-2\cos(3t_0) - 3\sin(3t_0)) = 0$ → $\tan(3t_0) = -2/3$ → first positive solution: $t_0 = (\pi - \arctan(2/3))/3 ≈ 0.73$ rad.
Minimum E[Z²] ≈ 8(1 − e^{-1.46}·cos(2.19)) ≈ 8(1+0.10) ≈ 8.8 W. As t₀→0: E[Z²]→0 (trivially). As t₀→∞: E[Z²]→8 (process decorrelates). The minimum for t₀>0 occurs where the cos term achieves its most negative value.
D
Chebyshev Bound — Second Moment Version
With E[X(t)]=3, Var(X)=E[X²]−(E[X])²=13−9=4, σ=2.

$P(|X(t)| \geq k) = P(|X(t)-3+3| \geq k)$. Using second-moment Markov: $P(|X|\geq k) \leq E[X^2]/k^2 = 13/k^2$.
Set 13/k² ≤ 0.1 → k² ≥ 130 → k ≥ √130 ≈ 11.4.

Alternatively using Chebyshev centered at μ: P(|X−3|≥k−3) ≤ 4/(k−3)² ≤ 0.1 → k−3 ≥ √40 → k ≥ 3+√40 ≈ 9.3. Choose the tighter approach depending on whether you want |X|≥k or |X−μ|≥k. Both are valid Chebyshev applications.
E
Is Poisson WSS? — Conceptual Mastery
No — N(t) is NOT WSS.

Proof: WSS requires constant mean. But E[N(t)] = λt, which depends on t. Therefore N(t) is not WSS.

Var(N(t)) from R_N: Var(N(t)) = R_N(t,t) − (E[N(t)])² = λt + λ²t² − λ²t² = λt. This confirms Var = Mean = λt.

Deeper insight: N(t) has stationary independent increments — the distribution of N(t+s)−N(t) depends only on s. This is a different property from WSS of the process itself. Confusing these two is a classic error. A process can have stationary increments but not be a stationary process. The increments Δ_s(t) = N(t+s)−N(t) for fixed s form a stationary process; N(t) itself does not.
