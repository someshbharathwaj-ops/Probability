⚡ Intuition

The autocorrelation R(τ) = E[X(t)X(t+τ)] measures how much the process remembers its own past. When τ=0 it equals the total power. As τ→∞, memory fades—decorrelation is forgetting. The oscillatory decay you see is the signature of a damped harmonic random process: a physical system being driven by noise.

⚡ The Deep Fact

The Poisson process has a beautiful property: its mean equals its variance — both equal λt. This is not a coincidence; it is the defining character of Poisson randomness. When you see a counting process where the "noise energy" equals the "average level," you are witnessing Poisson structure.

God-Level Problem · Level III

The Monster Problem:
When All Concepts Collide

This problem integrates stationarity, autocorrelation, PSD, and the Chebyshev bound. It initially appears underdetermined. The insight is that the constraints are stronger than they first appear.

Problem. Let \(X(t)\) be a wide-sense stationary (WSS) random process with autocorrelation function: \[ R_X(\tau) = \sigma^2 e^{-\lambda|\tau|}\cos(\omega_0 \tau) \]
(a) Without computing any transform, argue from first principles why \(R_X(\tau)\) is a valid autocorrelation function.

(b) A "filtered" version of \(X(t)\) is defined as: \[ Y(t) = X(t) - \alpha X(t - t_0) \] for constants \(\alpha \in \mathbb{R}\) and \(t_0 > 0\). Determine all values of \(\alpha\) and \(t_0\) such that \(Y(t)\) has zero power at the origin of its PSD, i.e., \(S_Y(0) = 0\).

(c) For the specific case \(\omega_0 = \lambda\), use the Chebyshev inequality to bound the probability that \(|Y(t)| \geq k\sigma\) for a given \(k > 0\), expressing your bound in terms of \(\alpha\), \(k\), \(\lambda\), and \(t_0\).

(d) Conceptual twist: A colleague claims that if \(R_Y(0) = 0\), then \(Y(t) = 0\) almost surely. Is this correct? Prove or disprove.
①

Intuition First — What Does Validity Mean?

A function is a valid autocorrelation if and only if it is symmetric (since \(R(\tau) = R(-\tau)\)) and positive semi-definite. The second condition means: for any set of times \(t_1,...,t_n\), the matrix \([R(t_i-t_j)]\) has non-negative eigenvalues. Rather than verify this algebraically, observe: \(R_X(\tau) = \sigma^2 e^{-\lambda|\tau|}\cos(\omega_0\tau)\) is the real part of \(\sigma^2 e^{-(\lambda - i\omega_0)|\tau|}\). Its PSD (via Wiener-Khinchin) must be non-negative for it to be valid. We verify this in step ②.

②

Compute the PSD via Wiener-Khinchin

\[ S_X(f) = \int_{-\infty}^{\infty} R_X(\tau) e^{-j2\pi f\tau} d\tau \] Using the Fourier pair \(e^{-\lambda|\tau|} \leftrightarrow \frac{2\lambda}{\lambda^2 + (2\pi f)^2}\) and the frequency shift property: \[ S_X(f) = \frac{\sigma^2\lambda}{\lambda^2 + (2\pi f - \omega_0)^2} + \frac{\sigma^2\lambda}{\lambda^2 + (2\pi f + \omega_0)^2} \] This is a sum of two Lorentzians — clearly non-negative for all \(f\). Validity confirmed. The physical meaning: this is the spectrum of a damped oscillator driven by white noise, centered at \(\pm\omega_0/2\pi\) with bandwidth \(\lambda\).

③

Part (b) — Zero Power at Origin

First, \(R_Y(\tau) = E[Y(t)Y(t+\tau)]\). Since \(X\) is WSS, \(Y\) is also WSS. Expanding: \[ R_Y(\tau) = (1+\alpha^2)R_X(\tau) - \alpha R_X(\tau+t_0) - \alpha R_X(\tau - t_0) \] The PSD of \(Y\) is: \(S_Y(f) = |1 - \alpha e^{-j2\pi f t_0}|^2 S_X(f)\)
Setting \(S_Y(0) = 0\): we need \(|1 - \alpha e^0|^2 = 0\), i.e., \(\alpha = 1\).
The insight: the condition \(S_Y(0)=0\) forces \(\alpha = 1\) for any \(t_0 > 0\). The filter \(Y(t) = X(t) - X(t-t_0)\) is a differencing operation that removes the DC component — a high-pass filter whose zero is at \(f=0\).

④

Part (c) — Chebyshev with ω₀ = λ

With \(\alpha = 1\), \(E[Y(t)] = 0\) (inherited from \(X\)). The variance: \[ \text{Var}[Y] = R_Y(0) = 2R_X(0) - 2R_X(t_0) = 2\sigma^2(1 - e^{-\lambda t_0}\cos(\lambda t_0)) \] By Chebyshev: \(P(|Y(t)| \geq k\sigma) \leq \frac{\text{Var}[Y]}{k^2\sigma^2} = \frac{2(1 - e^{-\lambda t_0}\cos(\lambda t_0))}{k^2}\)
Geometric insight: the bound is tightest when \(e^{-\lambda t_0}\cos(\lambda t_0)\) is maximized, which occurs at \(t_0 = 0\) (trivially, where \(Y=0\)) and has its first positive maximum between \(0\) and \(\pi/\lambda\). The Chebyshev bound measures how much power the differencing operation preserves.

⑤

Part (d) — The Conceptual Trap

The colleague is wrong. \(R_Y(0) = E[Y^2(t)] = 0\) implies \(E[Y^2(t)] = 0\), which means \(Y(t) = 0\) in mean square, i.e., almost surely. So the statement is actually correct—but the reasoning must be precise: \(E[Z^2] = 0 \Rightarrow Z = 0\) a.s. for any real random variable \(Z\), since \(Z^2 \geq 0\) and its expectation vanishes.
The trap: students often confuse "zero autocorrelation at all lags" with "zero second moment." Here, \(R_Y(0) = 0\) alone (not requiring all lags) is sufficient to conclude \(Y(t) = 0\) a.s., because \(R_Y(0)\) IS the second moment.
